# OptiProfit AI (Light) – Projektkontext

## Ziel
B2B-Tool für Getränkemärkte: Preise & Aktionen prüfen, maximal ROI, minimal Komplexität.

## MVP (Expo Mobile)
- Eingabe als Text (Chat-Style)
- Parser extrahiert: item, EK, VK, optional VK_normal, optional WKZ_total, optional Menge
- App rechnet deterministisch (ohne Pfand)
- Ergebnis-Karten + Historie lokal (AsyncStorage)

## Nicht im MVP
- Pfand
- Kassenschnittstelle
- Login/Accounts
